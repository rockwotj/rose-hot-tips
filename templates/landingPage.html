Rose-Hulman Hot Tips




Insider information on your classes





To view ALL courses in the selected term, enter a blank search.
To view all courses with a particular prefix in the selected term, enter the course prefix in the search bar.
Examples: MA in the search field will return all courses and sections with the MA prefix;
MA111 will return all sections of only MA111; MA2 will return all 200-level MA courses.
To view multiple course prefixes in the selected term, enter desired course prefixes, separated by a pipe, such as MA|PH.
